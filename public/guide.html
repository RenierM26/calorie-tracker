Calorie & Weight Tracker — Quick Guide

What this app does

  • Log meals with type, quantity (kg), calories, notes, and tags
  • Log weight daily
  • Show trends and graphs over selectable date ranges
  • View and edit full meal history in the Journal page

How to log meals

  1. Open the dashboard
  2. Fill date, type, food notes, kg, and calories
  3. Optionally add tags (e.g. starter, dessert, soda)
  4. Click Save Meal

You can log multiple entries for the same meal (starter/main/dessert/drinks).

How to log weight

  1. Set date
  2. Enter weight in kg
  3. Click Save Weight

Tip: log once daily at a similar time for better trend consistency.

Using date ranges

  • Choose presets: 7, 14, 30, or 90 days
  • Or choose a custom start and end date
  • Click Apply Range to update summary + charts

Meal Journal page

Use the Journal page to review all meals by date and filter by:

  • From/To dates
  • Meal type
  • Search text (food notes or tags)

Meals can be edited or deleted directly from Journal.

Editing and deleting

  • From dashboard: recent items have Edit and Delete
  • From journal: every meal row has Edit and Delete

Delete is permanent.

Export to Excel

  • On Dashboard: export Meals and Weights as Excel-friendly CSV files
  • On Journal: export the currently filtered list as CSV

Open CSV files directly in Excel for analysis and custom charts.

Need help?
