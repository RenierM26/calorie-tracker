# Calorie & Weight Tracker — User Guide

This app helps you:
- log meals (type, kg, calories, notes, tags)
- log body weight
- view trends over time
- edit/delete entries when needed

## Dashboard

Main page: `http://localhost:8092`

### Add Meal
1. Pick date and meal type
2. Add food notes (optional)
3. Add tags (optional, like `starter, dessert, soda`)
4. Enter quantity in kg
5. Enter calories
6. Click **Save Meal**

You can add multiple entries for the same meal period (starter/main/dessert/drink).

### Add Weight
1. Pick date
2. Enter weight in kg
3. Click **Save Weight**

Tip: one weight entry per day at a consistent time gives better trends.

## Date Ranges and Graphs

Use **Date Range** to:
- choose preset windows (7/14/30/90 days)
- or select custom start/end dates

Click **Apply Range** to update summary cards and charts.

## Journal Page

Journal: `http://localhost:8092/journal.html`

Use filters for:
- date from/to
- meal type
- search text (food or tags)

Edit and delete meals directly from each row.

## In-app Guide

Guide page: `http://localhost:8092/guide.html`

Accessible from dashboard and journal pages.
